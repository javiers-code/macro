# KEY spammer:
Is a macro tool made with python.

# How to use:
  To use the macro, run
  ```
  git clone "https://github.com/javiers-code/key-spammer"
  ```
  Then, cd into the directory
  ```
  cd key-spammer
  ```
  Now, run the python file
  ```
  python3 main.py
  ```
